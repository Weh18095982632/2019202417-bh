3 游戏机制
3.1、两种类型的敌人:普通和boss boss血更厚速度更快
3.2、防御塔可升级，可拆除(左键升级，右键拆除)
	有3种防御塔，一个范围减速，一个范围攻击，一个单体攻击
3.3、难度随波数提升而增加，敌人速度更快，血量更高。
3.4、防御方分值(资源)累计，可用于升级防御塔，杀死敌人资源数提高
3.5、终极技能，可以对敌人造成aoe伤害和减速。cd机制可时间cd，也可获取一定资源后可释放
